# ProyectoStarWars
Realizado por Javier Vázquez Cabello y Daniel Morón Márquez

La página web consiste en cuatro secciones en las que podemos seleccionar una nave y ver sus detalles, crear, editar, eliminar y actualizar pilotos. También, podemos crear, mover y eliminar misiones y, por último, ver un resumen con las distintas estadísticas de pilotos, naves y misiones.
